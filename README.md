#React Course Goal app

A basic application  built with React, JavaScript, and CSS.

-----

##### Download:

`$ git clone https://github.com/samira1992/react-course-goal.git`

`$ cd path to the folder`

`$ npm install `


-----------
### Running the code
`npm start`
